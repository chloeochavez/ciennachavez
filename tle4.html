Lesson 1

From Lesson 1￼, I learned that JavaScript is one of the most popular programming languages used to make websites more interactive and dynamic instead of just plain and static. I understood the difference between static and dynamic websites, and how JavaScript helps websites respond to user actions like clicking buttons or loading a page. I also learned that JavaScript was created by Brendan Eich and that it runs on the user’s computer. It can validate forms, create cookies, and add cool effects to websites. Lastly, I learned the difference between HTML, CSS, and JavaScript, HTML makes the structure, CSS designs it, and JavaScript makes it interactive.

Lesson 2

From this lesson, I learned about where we can place JavaScript in an HTML file. I learned that scripts can be placed in the  section if we want them to load before the page shows, or in the  section if we want them to run when the page loads. I also learned that we can use both head and body for scripts, and that we can create external JavaScript files using the .js extension so we don’t have to rewrite the same code again and again. I learned how to use comments in JavaScript and important guidelines like JavaScript being case-sensitive and how white spaces help make the code easier to read.

Lesson 3

In this lesson, I learned more about the characteristics of JavaScript, like how it is an interpreted, scripting, client-side, and object-oriented language. I learned that JavaScript works together with HTML and runs in the browser. I also learned about objects, methods, properties, events, and parameters, and how they all work together. I understood that objects like window, document, math, and navigator have different methods and properties that we can use in coding. I also learned about JavaScript syntax rules, how it is case-sensitive, and how scripts are processed from top to bottom.

Lesson 4

From this lesson, I learned about JavaScript events and how they are actions like mouse clicks, keyboard presses, loading a page, or submitting a form. I learned different event triggers like onclick, onload, onsubmit, onmouseover, and more. I also learned about popup boxes such as alert boxes, prompt boxes, and confirm boxes, and how they are used to show messages, get user input, or ask for confirmation. Lastly, I learned about variables, the rules in naming them, different data types like Boolean, and basic arithmetic operators used in JavaScript.
